# 실업급여.info

2026년 기준 실업급여(구직급여) 정보 포털. 고용보험법 및 고용노동부 공식 자료 기반.

## 페이지 구성

| URL | 설명 |
|-----|------|
| `/` | 실업급여 계산기 메인 |
| `/eligibility/` | 수급 자격 조건 5가지 |
| `/voluntary-quit/` | 자발적 퇴사 예외사유 13가지 |
| `/apply/` | 신청 방법 단계별 가이드 |
| `/faq/` | 자주 묻는 질문 20가지 |

## 기술 스택

- 정적 HTML / CSS / Vanilla JS
- 외부 의존성 없음 (Google Fonts CDN 제외)
- Netlify 호스팅

## 2026년 주요 수치

- 1일 상한액: 68,100원
- 1일 하한액: 66,048원
- 수급 기간: 120~270일

## 출처

- 고용노동부 공식 블로그
- 고용보험법 제40조·제58조 (국가법령정보센터)
- 고용보험법 시행규칙 [별표 2]
- 고용보험 공식 사이트 (ei.go.kr)
- 찾기쉬운 생활법령정보 (easylaw.go.kr)
